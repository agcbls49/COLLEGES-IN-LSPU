# LSPU Colleges in Java
A Java Swing program that shows details about these specific colleges (CCS, CBAA, COE, CIT and CTE), in Laguna State Polytechnic University San Pablo Campus. It includes a Change Theme button that changes all the themes on each frame randomly and a Revert Theme button to keep the original styling for each college. The Change Theme button has 4 different themes that applies randomly on each frame.
